### Usage 
We run the code with torch version: 1.7.0, python version: 3.8.5
* Train CIFAR100
```
python main.py
```
* Train Tiny-ImageNet
```
cd Tiny-ImageNet
python main_tiny.py
```
* Train ImageNet-Subset
```
cd ImageNet-Subset
python main_PSSR_imagenet.py
```
